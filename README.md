# TradingRobot
# 📈 혼합 전략 트레이딩 봇 (Swing + Trend)

---

## 개요

이 프로젝트는 **장기 추세 추종 전략과 단기 과매도 매수 전략을 통합**한 자동화 트레이딩 봇입니다.

* 장기 상승 추세 종목만 선별
* 단기 과매도(RSI 기반) 시 진입
* 장기 추세 이탈 시 자동 매도
* 포트폴리오 관리, 로그 기록, 이메일 알림 포함

---

## 주요 기능

1. **자동 종목 스크리닝**

   * Finviz API를 활용하여 과매도 및 시가총액 조건에 맞는 후보 종목 선별

2. **단기/장기 신호 분석**

   * 단기: RSI(2) + SMA5/SMA10 교차 → 반등 매수 기회 포착
   * 장기: SMA20/SMA50 교차 → 추세 판단 및 매도 결정

3. **포트폴리오 관리**

   * 보유 종목 기록 및 제거
   * 중복 신호 방지 및 하루 1회 알람 기록

4. **자동 이메일 알림**

   * 매수/매도 신호 발생 시 Gmail을 통해 실시간 알림 전송

5. **로그 관리**

   * 모든 매수/매도 이벤트와 신호 발생 기록

---

## 기술 스택

* Python 3.11
* 주요 라이브러리: `yfinance`, `pandas`, `ta`, `smtplib`, `finvizfinance`
* 개발 환경: Raspberry Pi 또는 일반 Python 환경

---

## 사용 방법

1. `portfolio.txt`와 `trading_bot_log.txt` 파일 생성
2. Gmail 계정 설정 (발신/수신)
3. Python 실행:

```bash
python trading_bot.py
```

4. 매수/매도 신호 발생 시 이메일 확인 가능

---

## 프로젝트 특징

* **하루 만에 구현 완료**: 단기간에 전략 설계 + 자동화 구현 능력 입증
* **실전 감각 반영**: 결측치 처리, 오류 처리, 포트폴리오 관리 포함
* **확장 가능**: 전략 파라미터 변경, 백테스트, 추가 지표 통합 용이
